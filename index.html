Phenotype Installer - Warning

If you can read this text in your webbrowser, your host configuration ist most probably not adjusted as intended.


The root path of your domain, subdomain, vhost or whatsoever should point to the directory htdocs within the phenotype folder structure.


We did that to assure, that neither any phenotype system file nor any of your application files (like templates, custom php code) is accesible via http). Nevertheless you can continue with the installation.
